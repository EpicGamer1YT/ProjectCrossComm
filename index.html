Welcome to Project CrossComm!|

Simple. Fast. Safe. Reliable.

Project CrossComm is a free-of-cost chatting program that runs in any browser (Chrome, Firefox, Safari, etc.). All of the chats are encrypted, so no one can tap into your chats. CrossComm is a multipurpose chatting system:

  • Use CrossComm for work!
  • Use CrossComm for school!
  • Use CrossComm at home!
  • CrossComm can be accessed and utilized almost anywhere!

Enjoy the program and chat away!
